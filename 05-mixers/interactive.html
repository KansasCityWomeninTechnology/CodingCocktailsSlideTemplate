Repeat some code as long as some condition is true


Types of loops:

  • for
  • for...in
  • for...of

Objects

Objects: how to make one



  • Create an object with 'new Object()' or initialize an empty one with '{}'
  • Use JSON (JavaScript Object Notation) to initialize
  • 'key':'value' pairs -- "key" may also be referred to as 'name' or 'property'

Ada Lovelace


Objects: how to make one

var AdaLovelace = {
                "achillesHeel": "Compulsive Gambling",
                "firstName": "Ada",
                "lastName": "Lovelace",
                "children": ["Byron","Ralph","Anne"],
                "dateBorn": "12/10/1815",
                "dateDied": "11/27/1852",
                "father": "Lord Byron",
                "Likes Programming": true,
                "fullName": function() {
                return this.firstName + " " + this.lastName;
                }
};

Objects: how to get data


  • You can get data from the Object by the key
  • Dot Notation
    • console.log(AdaLovelace.children)
// ['Byron', 'Ralph', 'Anne']
    console.log(AdaLovelace.fullName)
/* function () { return this.firstName + " "
+ this.lastName; } */
    console.log(AdaLovelace.fullName())
// Ada Lovelace

Objects: how to get data


  • Property name has to be alphanumeric (but can contain underscores)
  • Can't start with a number
  • No spaces
    • console.log(AdaLovelace.Likes Programming)
// syntax error :(

Objects: how to get data


    Bracket Notation

  • You can get data from the Object using the key

    Templates



    • A template is a reusable chunk of code used for markup
    • It contains placeholders (of variables) for where the dynamic content is populated

    How to use Templates



      Framework

      • A full suite of tools & helpers
      • Frameworks have their own way of templating


      React

      Angular

    How to use Templates



      Library

      • Set of tools that helps you write JavaScript faster
      • Provides more flexibility in terms of architecture


      Handlebars.js

      Mustache.js

      doT.js

    How to use Templates



      Native JavaScript

      • With ECMAScript 2015, we now have template literals
      • Provides an alternative to concatenating strings, which allows for us to explore templating with native JS
      • We'll be using this tonight

    Template literals


    var caliberOfDay = 'sunshiney';
    
var myMultiLineString = `
Hello!
Hope you have a ${caliberOfDay} day!`;

    VS

    var myMultiLineString = 'Hello!' + \n
'Hope you have a ' + caliberOfDay + ' day!';

    Template literals



    Expression with some logic in a template



    `I should drink ${8 - 5} more glasses of water today.`
// I should drink 3 more glasses of water today.
