Introduction

What is the command line?


Command Line Interface


Text-based interface
to interact with your computer


Enter input and return output


Why use the command line?

Power.


Easily run programs


Write scripts for automated tasks


A single command can run a task
that can save you hours


Running many of these commands together
can save you days

With Great Power...

comes great responsibility


You can tell the computer to do
anything that it is capable of


You can erase all of your data
with a simple command


Make sure you know what a
command does before you run it

How do I use it?


You need a terminal emulator:

  • Mac: iTerm + oh-my-zsh
  • Windows: Git Bash

Basic Commands

Basic Commands


pwd print working directory
ls list files & directories
cd change directory
mkdir make directory
rmdir remove directory
touch create file
rm remove a file

More commands


cp copy file or directory
mv rename/move file or dir
vim edit a file
echo print a message
ping check internet status
alias make custom shortcuts
grep global search

Helpful Tips

Tips to help you

Display usage docs

Abort a command CTRL + C
Clear terminal clear
Buffer recall up and down arrows
History history
Find prev command CTRL + R
Repeat last command !!
Attempt autocompletion hit TAB
Path population drag & drop the file

More Power: Scripts


  1. What is a script?
    1. A plain text file containing a series of commands
    2. Convention is to end file in `.sh` extension
    3. Great for:
      1. Sequence of commands you have repeated use for
      2. Setting up automated tasks
