  • Component-Based Architecture

  • Virtual DOM

1Ô∏è‚É£ Component-Based Architecture



  • Apps built from small, reusable pieces

  • Example: Netflix

  • Gives us Vertical Separation of Concerns

  • Makes apps faster and smoother

  • i.e. You tell React what the UI should look like...
