The Basics

What is HTML?



Hyper Text Markup Language


Most fundamental building block of web pages


Describes how the content is displayed


Indicates structure, not style (font, color, size)

How do you write HTML?



Create a .html file


HTML is a markup language


Markup and code is written/edited
in an IDE (like Atom)


Integrated Development Environment

Why use an IDE?



Syntax highlighting helps you "see" the code
with colors & other visual cues


Linters help you avoid mistakes
by highlighting them


Auto-complete "guesses"
what you might want to type next

HTML Syntax


HTML is a standard set of "tags"



Open/Close tags <p>Hey girl</p>


Self-Closing tags

HTML Structure



Every HTML page requires 4 tags:

The Document Object Model

What Browsers Do


  1. Download the HTML file (Server -> Client)
  2. Parse the HTML (code -> DOM)
  3. Load and apply CSS
  4. Load and apply Javascript
  5. Render the DOM visually *

What could possibly go wrong?


  1. Can't find HTML file on server
  2. Parse error: Missing end tag
  3. Can't find CSS file on server
  4. Can't find JS file on server
  5. Parse error in those files

When you're stuck


Pay attention to your IDE
(syntax highlighting and linter warnings)


Use a validator


Look for missing end tags (indent!)
