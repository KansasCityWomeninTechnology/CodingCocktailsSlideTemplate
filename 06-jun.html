Data Structures

Data Structures

  • When you have a collection of data, you need to organize it
  • That's when you need Data Structures like Arrays & Objects
  • Data Structures are made up of various variable types, functions, and even other Data Structures

Arrays

Arrays: how to make one

When you want a collection of data items or even functions

var cocktails = ['Code on the Beach',
        'jQuery Sour'];
        var drinksIveConsumed = [0, 1, 2, 3, 4, 5, 6,
        7, 'none of the above'];
        var anArrayOfFunctions = [orderDrink, closeTab];
var myNewArray = new Array();
        var myEmptyArray = [];

Arrays: stuff you can do

  • You can grab items from the array using their index
  • Arrays have a Zero-Based Index
  • To get the first item you use 0, to get the last item you subtract 1 from the number of items in the array
var first = cocktails[0]
        // this will return 'Code on the Beach'
        var last = cocktails[cocktails.length - 1];
        // returns 'jQuery Sour'

Arrays: stuff you can do

You can loop over an array:

var cocktails = ['Code on the Beach',
        'jQuery Sour'];
        // array 'for loop'
        cocktails.forEach(function(cocktail,index)){
        console.log(cocktail,index);
        }
        // 'Code on the Beach' 0
        // 'jQuery Sour' 1

Arrays: stuff you can do

Add an item to the end of an array:

cocktails.push('Mojito');
        // cocktails now looks like this:
        // ['Code on the Beach', 'jQuery Sour', 'Mojito']

Remove an item from the end of an array:

cocktails.pop();
        // cocktails now looks like this:
        // ['Code on the Beach', 'jQuery Sour']

Last-In-First-Out aka LIFO

Arrays: stuff you can do

Add to the beginning of an array

cocktails.unshift('JavaScriptini');
        /* ['Javascriptini', 'Code on the Beach',
        'jQuery Sour']*/

Remove from the beginning of an array

cocktails.shift();
        // ['Code on the Beach', 'jQuery Sour']

Loops

Loops

  • Repeat some code
  • As long as some condition is true
  • Different kinds of loops:
    • while
    • do...while
    • for...in/of/statement
    • Today we will learn about the 'for' loop

for Loop

Pseudo Code

for (initialization; condition; increment) {
        // code to execute while condition is true
        }

Example

var numberOfCustomers = 5;
        for (var i = 0; i < numberOfCustomers; i++) {
        notifyDrinkIsReady(i);
        }

for Loop

var numberOfCustomers = 5;
        for (var i = 0; i < numberOfCustomers; i++) {
        notifyDrinkIsReady(i);
        }

one of these will never terminate & is an 'infinite loop' 

var numberOfCustomers = 5;
        for (var i = 0; i < numberOfCustomers;) {
        notifyDrinkIsReady(i);
