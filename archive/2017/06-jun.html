jQuery vs. JavaScript



jQuery is a library of pre-packaged JS functions (remember those?) that help you write your JavaScript code faster.

jQuery vs. JavaScript


Select the Element 'my-id'

using Javascript:

document.getElementById('my-id')

using jQuery:

$('#my-id')

jQuery vs. JavaScript


Select the Class Element 'my-class'

using Javascript:

document.getElementsByClassName('my-class')

using jQuery:

$('.my-class')

jQuery vs. JavaScript


Select Elements With a Specific Attribute Value

using Javascript:

document.querySelectorAll('[attribute="value"]');

using jQuery:

$("[attribute='value']")
