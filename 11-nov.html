Tonight's Agenda:
Code Jam

Work together to create
a team web site or web application!

Option 1


  • Start with a template

  • Change content and styles

  • Add more pages and navigation on how to get there

Option 2


  • Start with a scaffold

  • Add text and images

  • Style your page

  • Add some interaction using jQuery and/or JavaScript

  • Add more pages

Option 3


  • Start with either our trivia app from October or a new Angular2 project

  • Add another component - maybe navigation?

  • Modify the API

  • Add another route

Option 4


  • Start with an open source project

  • Find an issue to work on

  • Make the requested changes

  • Follow the contributing submission process
