Introduction

What is the command line?


Command Line Interface


Text-based interface
to interact with your computer


Enter input and return output


Why use the command line?

Power.


Easily run programs


Write scripts for automated tasks


A single command can run a task
that can save you hours


Running many of these commands together can save you days

With Great Power...

comes great responsibility


You can tell the computer to do
anything that it is capable of


You can erase all of your data
with a simple command


Make sure you know what a
command does before you run it

How do I use it?


You need a terminal emulator:

  • Mac: iTerm + oh-my-zsh
  • Windows: Cmder

Basic Commands

Basic Commands



pwd print working directory
ls list files & directories
cd change directory
mkdir make directory
rmdir remove directory
touch create file
rm remove a file

More commands


cp copy file or directory
mv rename/move file or dir
echo print a message
ping check internet status
grep global search

I'm lost. Send help...


You can always get back to home in one easy command.


Windows

cd %USERPROFILE%

Mac:

cd ~

What is Version Control?



A utility that tracks changes in files



  • Reference past changes
  • Restore previous versions

Why use Version Control?



Changes to source code are recorded


If a mistake is made, you can rollback changes




  • Version History in Google Docs
  • Time Machine on MacOS
  • Undo in any text editor/processor

Creating a Repo & Adding New Files


git init

Cloning a Repository



git clone [url]

Git Term: origin


When the clone is pulled down to your machine, it keeps a reference to the parent, which the default term in git is origin

Essential Git Commands



git pull

Fetches and merges: Updates references & merges those updates into your local repo


git fetch

Updates references in your local repo, but does not change any files

Branches





Credit: Atlassian Git Tutorial

Branch Related Commands



Assuming you are on master branch...


git checkout -b MyFeatureBranch

Creates and automagically switches to MyFeatureBranch


git checkout master

Switches back to the master branch

Branch Related Commands



git checkout MyFeatureBranch

Checks out the branch that we created


git checkout -b MyOtherFeatureBranch

Creates and automagically switches to MyOtherFeatureBranch - except it would have MyFeatureBranch as it's base

Making Changes Locally


git status

Shows information about status, including staged and unstaged changes

git add [filepath]

git add [dir]

git add .

Add a file or directory (untracked or modified) to staging

Making Changes Locally


Oops I didn't mean to stage that...

git reset

Resets (undoes) a `git add`



Yep I like my changes and want to keep them...

git commit -m "[message]"

Stores your current contents of the index in a new commit

git log
