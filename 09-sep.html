Back end development

What is it?


Serves pages to the browser


Receives input: validates, manipulates, & then...


Interacts directly with storage [database/files]


Languages:
Java, PHP, Python, C++,
C#, .NET, Node.js & more


Why Node.js?


Back-end Javascript


Javascript runtime is fast


Built on Chrome's V8 JS engine,
written in C++


Cross-platform: Windows, Mac & Linux

APIs

What are they?


Application Programming Interface

A service to go between two different programs


Can directly interact with the database


Can be defined in many ways

Hyper Text Transfer Protocol

SOAP vs REST



SOAP


  1. Simple Object Access Protocol

  2. Older, standardized (strict) communication protocol

  3. Only supports a superset of HTML, eXtensible Markup Language

  4. POST for everything

REST


  1. Representational State Transfer

  2. A newer, flexible architectural style

  3. More than 70% of public APIs

  4. Supports many formats

  5. GET, POST, PUT, DELETE

Paths, Routes & Endpoints

https://api.example.com/v1/users/6437
https://api.example.com/v1/users?role=admin&active
\________________________/\____/ \______________________/
       server URL        endpoint    params (optional)
                        path/route

Path parameters:

  1. - after a noun, separated by /
  2. - change the route

Query parameters:

  1. - start with ?, each separated by &
  2. - do not affect the endpoint
