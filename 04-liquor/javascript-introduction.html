functions in the Wild


Name your function
(declare a variable)


var orderDrink = function(drinkName) {};

Read as: The variable 'orderDrink' is a function with one parameter, called 'drinkName'

functions in the Wild


The task is specified inside the curly braces:

var orderDrink = function(drinkName) {
    return 'I would like a ' + drinkName +
    ' please.';
};

When you run the code, you decide what the value of the parameter (drinkName) is and insert it, Mad Libs style!

Functions in the Wild



Call the function by telling it what to use for the parameter:


orderDrink('gin and tonic');

Arrays are tricksy


Zero-based Index

Accessing Array Data

var fruit = ['Apple','Strawberry','Blackberry'];
                
// Grab first item and display
var first = fruit[0];
console.log(first);
// 'Apple'
                
// Check array length and display
var length = fruit.length;
console.log(length);
// 3
        
// Grab last item and display
var last = fruit[fruit.length-1];
console.log(last);
// 'Blackberry'

Arrays: stuff you can do

Add item to beginning of array

// Create the Array
var fruit = ['Apple','Pear','Kiwi'];
                    
// Add item 'Banana' to beginning of array
fruit.unshift('Banana');
            
// View the contents of the array
// stored in the fruit variable
console.log(fruit);
// ['Banana','Apple','Pear','Kiwi']

Arrays: stuff you can do

Add item to end of array

// Create the Array
var fruit = ['Apple','Strawberry','Kiwi'];
                    
// Add item 'Cherry' to end of array
fruit.push('Cherry');
                    
// View array contents
console.log(fruit);
// ['Apple','Strawberry','Kiwi','Cherry']

Arrays: stuff you can do

Remove item from start of array

// Create an Array
var fruit = ['Apple','Peach','Pear'];
                    
// Remove item from beginning of array
var item = fruit.shift();
// 'Apple'
                    
// View array contents
console.log(fruit);
// ['Peach','Pear']
                    
var length = fruit.length;
// 2

Arrays: stuff you can do

Remove item from end of array

Loops

Loops

What is a loop?

Repeat some code as long as some condition is true


Types of loops:

  • for
  • for...in
  • for...of
  • while
  • do...while

for Loop

for (initialization; condition; increment) {
// code to execute while condition is true
}

Example

var numberOfCustomers = 5;
                    
for (var i = 0; i < numberOfCustomers; i++) {
    notifyDrinkIsReady(i);
}

for Loop


var numberOfCustomers = 5;
for (var i = 0; i < numberOfCustomers; i++) {
    notifyDrinkIsReady(i);
}

Which of these is an 'infinite loop'? 

var numberOfCustomers = 5;
for (var i = 0; i < numberOfCustomers;) {
    notifyDrinkIsReady(i);
}

foreach


An array method that executes a given function once for each array element


// Create array
var drinks = ['martini', 'soda', 'coffee'];

// loop using forEach
drinks.forEach(function(drink) {
    console.log(drink);
});
// martini
// soda
// coffee
