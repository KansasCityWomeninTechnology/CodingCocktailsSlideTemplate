    APIs: What are they?


    Application Programming Interface

    A service to go between two different programs


    Can directly interact with the database


    Can be defined in many ways

    Hyper Text Transfer Protocol

    Swagger


    A set of tools that helps you design, build, document and consume REST APIs



    • Available endpoints
    • Operations on each endpoint
    • Operation parameters
    • Authentication methods
    • See the Swagger docs for our Trivia API

    Services


    A reuseable chunk of code that provides a specific service


    A service that fetches usernames

    A service that performs a complicated search algorithm

    Use the CLI to make a service for you:
    ng generate service drinks

    Put it together



    Create your service



    Consume your service


    Component uses the service to get the data

    Template displays the data

    The Drinks Service

    
            import { Injectable } from '@angular/core';
            import { HttpClient } from '@angular/common/http';
            import { Observable } from ‘rxjs’;
            import { Drink } from './drink.model';
            
            @Injectable({
              providedIn: ‘root’
            })

            export class Drinks {
            
              constructor(private http: HttpClient) { }
            
              getDrinks(): Observable<Drink[]> {
                return this.http.get<Drink[]>('app/drink-menu.json');
              }
            }

    The Component

    
            class DrinkComponent {
              public drinks: Observable<Drink[]>
           
              constructor(private drinkService: DrinkService) {
               this.drinks = this.drinkService.getDrinks();
             }
           }

    The Template

    
            <ul>
              <li *ngFor="let drink of drinks | async">
                {{drink}}
              </li>
            </ul>
