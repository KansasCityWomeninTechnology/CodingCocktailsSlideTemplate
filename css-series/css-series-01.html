What is CSS?


CSS stands for Cascading Style Sheets. It allows us to make the website look however we want!


So what can we change?



  1. Font Color
  2. Size
  3. Element placement on the page


We need CSS to make our website look pretty!

A few Examples!


Font Colors and Size

<p style="color: #484e78; font-size: 14px;">Hey girl</p>

General Size of Image

<img src="some/image/path" height="400" width="200">Hey girl</img>

Margin, Float and Padding

<p style="margin-top: 10px; margin-left: 10px;">Hey girl</p> <p style="padding-top: 5px; padding-left:6px;">Hey girl</p> <p style="float: left;">Hey girl</p>

How do we include CSS in a project?


Three ways to do it!

  1. Inline on HTML element.
  2. Internal to HTML file by adding to style tag on HTML
    <style>
        p {
            font-size: 15px;
         }
    </style>
  3. External CSS Files... Hint: this is the best practice!!

External CSS Files



CSS (AKA Stylesheets) have the file extension .CSS



When following best front-end practices .css files will live in a folder specifically for stylesheets



Example to add a stylesheet to our HTML



<link rel="stylesheet" href"styles/styles.css" >

The link tag links to an external file, in this case, is our stylesheet. The rel attribute tells the browser that the link is a stylesheet. The href tells the browser the location of the files.

You can use multiple stylesheets for one project. We can include as many links as we want. However, for this session, we will only use one.

Stylesheet General Information



First, a few quick vocabulary words:
Selector: the element weare creating the styles for.
Property: the part of the element we are styling (ie: color)
Value: the style we want (ie: for color we might use "blue")


    selector {
        property: value;
     }

Different Types of Selectors


Element Selectors - Styles all instances of that element

    h2 {
        color: green;
     }

Class Selectors - Styles all elements marked with that class

    .cocktail {
        color: pink;
     }
 <p class="cocktail">Hey girl</p>

You can also style elements differently based on element state. IE: We can change the look of an element if the user hovers over the element.

Class Naming Best Practices



Name classes based on what the class is not what the class looks like

  1. Allows for a clearer meaning of what the class is for.
  2. Makes maintaining the CSS easier.
  3. For responsive sites, (IE: sites that look different on mobile/tablet/desktop) how something looks may change between the various sizes. This would make a class based on how somethign looks make sense on one but not the other.

Units of Measurement


Absolute - the size is the size regardless of parent.

Pixels are the most common. There are others, but mostly will see pixels.


Relative - the size is depenedant on the parents size.

Percent - will make the element X% size bigger / smaller than the parent element.

rem/em - will make the element X times size bigger / smaller than the parent element.

vh/vw - will make the element take up X amount of the viewport height or width

Colors


Important note: Keep accessibility in mind when selecting colors. A big thing to note, is contrast being high enough. There are a lot of great tools to help with this.


RGB - rgb(Red, Green, Blue)


Each number passed into rgb is based on intensity. The higher the intensity, the more the color will be represented.

White: rgb(255, 255, 255)

Black: rgb(0, 0, 0)

Red: rgb(255, 0, 0)


Hexadecimal Numbers (#beb66d) - another way to represent colors.

https://htmlcolorcodes.com/color-picker/ can be used to get the hex value for any color!
