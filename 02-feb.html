What is CSS?



Cascading Style Sheets


Controls how elements look


Indicates style (font, color, size, position)


CSS3 is the latest version of CSS

Why do we care?

How do you use CSS?


Create a .css file


Ok, but now what?


Use any of the following 3 methods...

Method 1: Inline


<h1 style="color:orange;">My header</h1>

My header

Method 2: Internal

<!DOCTYPE html>
<html>
  <head>
    <style>
       h1 {color: orange;}
     </style>
   </head>
  <body>
    <h1>My header</h1>
  </body>
</html>

My header

Method 3: External

h1 {color: orange;} <!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
   </head>
  <body>
    <h1>My header</h1>
  </body>
</html>

Special Selectors

  1. Children (direct) ul > li { background-color: yellow; }
  2. Descendant (any) ul li { background-color: yellow; }
  3. Adjacent (immediate sibling) div + p { background-color: yellow; }
  4. Attribute a[target="_blank"] { background-color: yellow; }

Pseudo Classes

  1. /* unvisited link */
    a:link {
      color: #FF0000;
    }

    /* visited link */
    a:visited {
      color: #00FF00;
    }

    /* mouse over link */
    a:hover {
      color: #FF00FF;
    }

    /* selected link */
    a:active {
      color: #0000FF;
    }
