The Basics

What is CSS?

Controls how elements look

CSS3 is the latest version of CSS

Why do we care?

CSS Compilers

What is a CSS Compiler?



(a.k.a. pre- and post- processors)


Allows you to write faster & cleaner CSS


But Browsers only support CSS...

so you use a different syntax and compile it!

Pre-processed code -> CSS


Why Sass?


Several pre-processors to choose from

  1. - Less
  2. - Stylus
  3. - Sass


Each are slightly different (features & syntax)


Sass is the most common


Sass offers syntax flexibility based on file ext...

Sass file extensions


.sass

  1. - No curly braces or semi-colons
  2. - So indentation is important!
  3. - Older version of Sass


.scss ("Sassy CSS")

  1. - Looks more similar to CSS
  2. - Any file: .css -> .scss
  3. - Newer version of Sass

Sass Superpowers


  1. Variables - giving styles a name
  2. Nesting - define styles within another style
  3. Math - ceiling / floor, rounding, min / max
  4. @import - link one CSS file to another
  5. Mixins - perform computations on inputs
  6. Minification - compress the CSS like a zip
  7. Loops - repeat color / position changes
  8. @extend - style A plus style B

Sass -> CSS example

  1. Sass variable (source) $secondaryColor: #cb6f10;
  2. Sass (source) lighterDiv {
      background-color: $secondaryColor;
      border-top: 5px solid lighten($secondaryColor, 20%);
    }
  3. CSS (compiled) lighterDiv {
      background-color: #cb6f10;
      border-top: 5px solid #f1a250;
    }

Grouping Selectors

Special Selectors

  1. Children div > p { background-color: yellow; }
  2. Descendant div p { background-color: yellow; }
  3. Adjacent [Siblings] div + p { background-color: yellow; }
  4. Attribute a[target="_blank"] { background-color: yellow; }

Pseudo Classes

  1. /* unvisited link */
    a:link {
      color: #FF0000;
    }

    /* visited link */
    a:visited {
      color: #00FF00;
    }

    /* mouse over link */
    a:hover {
      color: #FF00FF;
    }

    /* selected link */
    a:active {
      color: #0000FF;
    }
