Dependencies, Packages & Package Managers

What is a dependency?


Assets that are required/used by other assets


  • Frameworks/Libraries
  • Your own files
  • Single file
  • Bundles of files (images & code)


Dependencies can have their own dependencies

What is a dependency?

Versioning: Lock Files

package-lock.json


Exact package versions locked down?

No! package-lock.json is synced with changes to package.json unless...


npm install moment --no-shrinkwrap
npm install moment --no-save

Be careful with manual updates to package.json!

Versioning: Lock Files

package-lock.json

{
  "name": "package-name",
  "version": "1.0.0",
  "lockfileVersion": 1,
  "requires": true,
  "dependencies": {
    "moment": {
      "version": "2.18.1",
      "resolved": "https://registry.npmjs.org/moment/-/moment-2.18.1.tgz",
      "integrity": "sha1-w2GT3Tzhwu7SrbfIAtu8d6gbHA8="
    }
  }
}
