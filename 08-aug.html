Dependencies, Packages & Package Managers

What is a dependency?


Assets that are required/used by other assets


  • Frameworks/Libraries
  • Your own files
  • Single file
  • Bundles of files (images & code)


Dependencies can have their own dependencies

What is a dependency?

<script src="assets/lib/jquery.bxslider/jquery.bxslider.min.js"></script>

Personal Projects: global install

Team Projects: project-only install

npm install browserify --save-dev

Browserify: Require Packages

  });
});

Browserify: Bundle

browserify main.js -o bundle.js

Use Watchify while developing

  "name": "my-package",
  "version": "1.0.0",
  "scripts": {

  }
