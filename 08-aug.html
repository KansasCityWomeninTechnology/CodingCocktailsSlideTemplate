Dependencies, Packages & Package Managers

What is a dependency?


Assets that are required/used by other assets


  • Frameworks/Libraries
  • Your own files
  • Single file
  • Bundles of files (images & code)


Dependencies can have their own dependencies

What is a dependency?

Updating Packages

Show out of date packages 
npm outdated
Update all outdated packages 
npm update
Update specific package 
npm update packageName
Update global packages 
npm update -g
Save updated versions to package.json 
npm update packageName --save
